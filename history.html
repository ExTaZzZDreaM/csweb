COUNTER-STRIKE: SOURCE

Counter-Strike: Source was released in 2004 as a remake of the original Counter-Strike. It was developed by Valve and uses the Source engine, which was also used in Half-Life 2. Counter-Strike: Source features updated graphics and physics, as well as new gameplay features such as dynamic lighting and ragdoll physics.

COUNTER-STRIKE: GLOBAL OFFENSIVE

Counter-Strike: Global Offensive was released in 2012 as the latest installment in the Counter-Strike series. It was developed by Valve and uses the Source engine, just like Counter-Strike: Source. Counter-Strike: Global Offensive features updated graphics and gameplay, as well as new features such as a competitive matchmaking system and a workshop where players can create and share custom content.

COUNTER-STRIKE ONLINE

Counter-Strike Online is a free-to-play version of Counter-Strike that was developed by Nexon for the Asian market. It was released in 2007 and features a number of changes from the original game, such as new maps, weapons, and game modes.

COUNTER-STRIKE: NEXON STUDIES

Counter-Strike: Nexon Studios is a free-to-play version of Counter-Strike that was developed by Nexon for the global market. It was released in 2015 and features a number of changes from the original game, such as new maps, weapons, and game modes.

Who developed Counter-Strike?

The first Counter-Strike, the mod for Half-Life, was developed by Minh Le and Jess Cliffe. Most Counter-Strike games are developed by Valve, but Counter-Strike: Condition was developed by Turtle Rock Studios, and Nexon develops the Asian versions of Counter-Strike.

What platform is Counter-Strike released on?

Counter-Strike games have historically been primarily PC games, releasing first for Windows and later for OSX and Linux. Counter-Strike: Global Offensive was also released for the Xbox 360 and PlayStation 3.

How does Counter-Strike differ from other first-person shooters?

Counter-Strike uses different mechanics than other first-person shooters. While most first-person-shooters include modes such as deathmatch (the number of kills determines the winner) or capture the flag (the winner brings the loser’s flag to their base), Counter-Strike’s assassination, hostage rescue, and bomb defusal modes–the most popular ones–require unique skills since the winner is not determined by simply killing everyone you see.

Does Counter-Strike allow mods?

Counter-Strike does allow players to mod the game to improve their own experience. It would be somewhat ironic if it didn’t, as the game was first spawned as a mod for Half-Life.

What was the prize pool of the first major Counter-Strike tournament?

The first major Counter-Strike tournament had a prize pool of $15,000.
